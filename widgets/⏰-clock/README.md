# ⏰ Clock Widgets

このフォルダには、**時刻表示を目的とした時計ウィジェット**をまとめています。  
Notion への埋め込みを前提とした構成です。

装飾性のある背景付き時計から、  
シンプル・ミニマムな時計まで、段階的に拡張していく予定です。

---

## 📌 基本の使い方
各フォルダ内の `index.html` を  
Notion に「埋め込み（Embed）」して使用します。

使用する URL は、各フォルダ内の README.md に記載しています。

---

## 📂 バリエーション構成

### scene
背景や世界観を含む、**情景重視の時計**  
（現在はこちらのみ実装済み）


### minimal（予定）
装飾を抑えた、**シンプル・ミニマムな時計**


---

## 🕒 表示形式（今後拡張予定）
- digital（デジタル）
- analog（アナログ）

※ 表示形式は、上記バリエーション配下で管理予定です。

---

## 📝 メモ
- 各最小単位フォルダには  
  `index.html` と `README.md` を配置
- 背景あり時計と、シンプル時計は思想を分けて管理

## 🌿 運用方針
- URL は手入力せず、README 経由で使用
