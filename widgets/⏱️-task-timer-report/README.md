# ⏱️ Task Timer Report

このフォルダには、**タスクタイマーの記録・レポート用途ウィジェット**をまとめています。  
計測結果の確認、ログ表示、業務日報への転記などを想定した構成です。

※ タイマー本体（計測機能）は  
`⏱️-task-timer/` フォルダにまとめています。

---

## 📌 想定用途
- タスクごとの作業時間の確認
- 開始時刻〜終了時刻の記録表示
- 業務日報・作業ログへの転記補助
- タイマー利用後の「振り返り」

---

## 📂 構成について
このフォルダ配下には、用途や表示形式ごとに  
複数のレポート系ウィジェットを配置予定です。

例：
- 日別ログ
- タスク別集計
- シンプル表示 / 詳細表示 など

---

## 📝 メモ
- 各最小単位フォルダには  
  `index.html` と `README.md` を配置
- 表示言語・形式ごとにサブフォルダで管理予定

---

## 🌿 運用方針
- タイマー本体とはフォルダを分離して管理
- URL は README 経由でコピーして使用
- Notion 管理では  
  `⏱️-task-timer/README.md` と並ぶ関連カテゴリとして扱う
