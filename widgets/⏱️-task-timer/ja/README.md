# ⏱️ タスクタイマー（日本語版）

このフォルダには、**日本語表示のタスクタイマーウィジェット**をまとめています。  
Notion への埋め込みを前提とした構成です。

表示サイズや配色の違いにより、複数のバリエーションがあります。

---

## 📌 使い方
各バージョンフォルダ内の `index.html` を  
Notion に「埋め込み（Embed）」して使用します。

URL は、各フォルダ内の README.md に記載しています。

---

## 📐 表示サイズ別バリエーション

### standard
標準サイズのタスクタイマー  

### slim
コンパクト表示のタスクタイマー  

### wide
横長レイアウトのタスクタイマー  

### single
1タスク専用のシンプルなタイマー  


---

## 🎨 配色バリエーション

### slim / mint-gray
Slim 版の配色違い（Mint Gray）  


※ 配色以外の動作・仕様は base 版と共通です。

---

## 🧩 バリエーションの考え方
- **standard / slim / wide / single**  
  → レイアウト・サイズの違い
- **base / mint-gray**  
  → 配色のみの違い

---

## 📝 メモ
- 各最小単位フォルダには  
  `index.html` と `README.md` を配置
- 英語版は `../en/` フォルダにまとめています

---

## 🌿 運用方針
- URL は手入力せず、README 経由で使用
- Notion 管理では  
  `⏱️-task-timer/README.md` を親リンク、  
  この README を日本語版の基準点として使用
