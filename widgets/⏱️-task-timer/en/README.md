# ⏱️ タスクタイマー（英語版）

このフォルダには、**英語表示のタスクタイマーウィジェット**をまとめています。  
Notion への埋め込みを前提とした構成です。

※ README は管理用のため、日本語で記載しています。

---

## 📌 使い方
各バージョンフォルダ内の `index.html` を  
Notion に「埋め込み（Embed）」して使用します。

使用する URL は、各フォルダ内の README.md に記載しています。

---

## 📐 表示サイズ別バリエーション

### standard
標準サイズのタスクタイマー  

### slim
コンパクト表示のタスクタイマー  

### wide
横長レイアウトのタスクタイマー  

### single
1タスク専用のシンプルなタイマー  

---

## 🎨 配色について
現在、英語版は **base（色なし）** のみです。  
配色違いは今後追加予定です。

---

## 🧩 バリエーションの考え方
- **standard / slim / wide / single**  
  → レイアウト・サイズの違い
- **base**  
  → 標準配色（動作仕様は共通）

---

## 📝 メモ
- 各最小単位フォルダには  
  `index.html` と `README.md` を配置
- 日本語版は `../ja/` フォルダにまとめています

---

## 🌿 運用方針
- URL は手入力せず、README 経由で使用
- 上位の  
  `⏱️-task-timer/README.md`
